# WebSite_Summarizer
Summarize any website using LLMs


Offline Website Summarization Tool powered by Ollama + DeepSeek.

## Features
- Website scraping
- Page-length controlled summary
- CLI & Streamlit UI
- Multi-model support
- Markdown export

## Run

streamlit run app.py
